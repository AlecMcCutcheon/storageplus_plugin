## StoragePlus 1.0.0

Initial release.

**Update:** Quick deposit now triggers on right-click (normal button push) instead of left-click.

## Minecraft version

- **Supported / tested:** 1.21.4
- **Spigot/Paper:** 1.21.4

## Compatibility

- Built for Spigot API 1.21.4. Use on 1.21.4 servers.

## Features

- **Labeled storage:** Floating labels on chests and barrels (name tag `[label] <name>`), double-chest support, open storage by right-clicking the label.
- **Quick deposit:** Right-click a button next to a barrel (or `/quickdeposit`) to deposit your inventory into nearby chests/barrels within 20 blocks.
- **Admin:** `/removearmorstands` to remove all armor stands (including labels). See `doc/` in the repo for full documentation.
